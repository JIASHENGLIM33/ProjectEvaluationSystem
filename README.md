---

##  Features

### Student
- Upload project deliverables: documents, slides, videos, GitHub links  
- View project submission status  
- View evaluation results when completed

### Evaluator
- View assigned projects
- Submit evaluation scores and written feedback  
- AI feedback aid for consistent evaluator comments

### Administrator
- Assign evaluators automatically or manually  
- Manage evaluator accounts (activate / deactivate)  
- Monitor evaluation workflow

---

##  System Architecture

PEMS is built using:
- **PHP** for backend logic  
- **MySQL** for database  
- **TailwindCSS** for frontend styling  
- **AI-assisted logic** for feedback suggestions
